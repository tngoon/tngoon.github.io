DSGN 90: Build Your Own Website or Portfolio


Course Time/Location: F 9-10:50am, HSS 1346
Office Hours: Tu 3-4pm @ the Design Lab, 1601 Atkinson Hall
Instructor contact: tngoon@ucsd.edu

In this course, you will learn basic technical and design skills to build your own website or portfolio. This seminar is also an opportunity for you to improve course materials for the upper-division Interaction Design (COGS 120) course and prepare for more advanced design courses. Bring your laptop to class each day to complete in-class assignments.

The skills you will learn are:

  • Source control with Git and Github
  • Basic debugging skills
  • HTML/CSS
  • Basic JavaScript

    • Each class session will consist of a lecture on concepts with activities throughout the lecture, followed by time to complete assignments in class.

    View syllabus

    Grading

    This course is a P/NP course. In order to pass, the following requirements must be met:

  • No more than 1 excused absence/tardy, no unexcused absences/tardies
  • 3 out of 4 assignments graded with a check or higher (on a check -, check, and check + scale)
  • Regular participation in class (answering questions, helping peers, participating in feedback sessions)

    • Criteria for assignment grading:

  • Check - : Assignment is incomplete and/or code contains major errors.
  • Check: Assignment is complete, no major errors in code.
  • Check + : Assignment goes above and beyond assignment requirements and material taught in class, no errors in code.
